# 企业信息化管理系统 GREEN - 交付包

版本：v1.0
日期：2026年4月

## 软件概述

GREEN 是一套面向制造业/工程类企业的信息化管理系统，功能覆盖：
- 项目与任务管理（Tasks）
- 审批流引擎（Approvals）
- 财务收支与发票（Finance）
- 客户与供应商关系管理（CRM）
- 设备与物料管理（Equipment/Material）
- 通知渠道集成（飞书/企微/钉钉/邮件）
- 操作审计日志
